# v1.11.0 — Fleet Labor Slice 2: the Agentic Executor

**Date:** 2026-07-09 · **PR:** #80 (squash `33c33c1`) · **Decision:** d078 · **Track:** "the fleet does the labor" (Slice 1 → v1.10.0)

The conductor's empty `-Spawner` seam is filled: **`/baton:go -Execute` makes the fleet
actually do the work.** An agentic instrument (codex, agy, claude-cli) edits a throwaway
git worktree; Baton proves labor happened by the diff growing, gates the produced diff
through the existing acceptance phase, and leaves branch `baton/run-<id>` for the human
to merge. Baton never merges.

## What's new

- **`scripts/fleet-executor-lib.ps1`** — worktree primitives (`New-RunWorktree`,
  `Get-RunDiff` with untracked-file capture, `Get-WorktreeTreeSha`, `Remove-RunWorktree`
  — branch always kept), `Test-ProviderAgentic`, and the `New-AgenticSpawner` factory
  (route → filter to edit-capable **fleet** providers → dispatch with cwd = worktree →
  proof-by-diff via `git write-tree`; nonzero exit fails the task; a throwing dispatch
  is contained, never crashes the walk).
- **`Invoke-Conductor -DiffProvider`** — optional post-walk seam: cumulative diff →
  `<run-dir>/changes.diff` → acceptance-gated. Absent ⇒ byte-for-byte unchanged.
- **`/baton:go --execute [--repo <path>]`** — run report names the branch and files
  changed: "N file(s) changed on branch `baton/run-<id>` — review and merge."
- **`agentic` fleet.yaml field** — authoritative when present; absent, inferred from
  `platform ∈ {claude, codex, gemini}`. Zero-config on the seed: codex / agy /
  claude-cli qualify; ollama/lm-studio/gh-copilot/github-models are filtered out of
  edit tasks (their diff-apply path is Slice 3).

## Manual smoke (box-private, pre- or post-deploy)

Like Slice 1's `--live` proof, the real round-trip is a box-private check, not a suite:

```powershell
# scratch repo, then:
pwsh -File "$HOME/.claude/scripts/fleet-go.ps1" -Goal "add a hello.md explaining this repo" -Execute -RepoPath <scratch-repo> -Json
# expect: status completed, branch baton/run-<id>, changes.diff carrying the new file
```

## Deferred (known, by design)

- Worktree GC: run worktrees under `<repo-parent>/.baton-worktrees/` persist until
  discarded (`Remove-RunWorktree`); the branch survives even that.
- Gitignored-only edits read as "no changes" (intended proof-by-diff semantics).
- Chat/local instruments emitting diffs Baton `git apply`s → **Slice 3**.
