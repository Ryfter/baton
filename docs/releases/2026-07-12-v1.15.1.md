## Visual overview (new)

Four SVG infographics under [`docs/assets/diagrams/`](../assets/diagrams/), embedded in the
README (and the command map in COMMANDS.md). Plain SVG — they render natively on GitHub in
both light and dark themes, no external dependency:

1. **Mental model** — the 3-tier Conductor (Conductor → Orchestrators → Instruments).
2. **How-to workflow** — the happy path, which command drives each step.
3. **Command map** — all 50 commands grouped into nine clusters.
4. **Fleet loop** — the route → dispatch → journal → learn/failover telemetry cycle.

Designed in Excalidraw, authored as SVG for durable inline rendering (the Excalidraw→PNG
export bridge dropped text labels; SVG renders faithfully and stays editable).
